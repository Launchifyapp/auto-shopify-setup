# Polaris App Home Examples

This directory contains example implementations demonstrating how to use Shopify Polaris App Home web components in TypeScript/Next.js.

## Files

### `polaris-app-home-example.tsx`

A comprehensive example showing various Polaris App Home components including:
- Page layout with heading and primary actions
- Sections for content organization
- Grid layouts with cards
- Buttons with different variants
- Typography components (heading, paragraph, text)
- Lists and dividers
- Stack layouts

This example demonstrates proper TypeScript usage with full type safety for all Polaris App Home custom elements.

## Usage

These examples are for reference only and are not part of the application build. You can:

1. **Copy code snippets** into your own pages/components
2. **Reference the patterns** when building your App Home pages
3. **Run the files** by importing them into your Next.js pages

## Type Definitions

All Polaris App Home web components are fully typed thanks to the declaration file at:
`/app/polaris-app-home-elements.d.ts`

This ensures that you get:
- ✅ Full IntelliSense/autocomplete support
- ✅ Type checking for component props
- ✅ No TypeScript compilation errors
- ✅ Better developer experience
